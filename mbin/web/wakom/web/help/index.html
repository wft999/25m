主页介绍

程式启动后即进入主页面，主页面主要用于显示机台概貌、AUOTO RUN重要数据以及为其他操作提供入口；画面如下图：

A:显示系统时钟；

B:显示系统当前警报信息，包括警报发生时间、警报码、警报内容；

C:显示系统当前登录用户；

D:显示系统当前处于手动或自动状态；

E:显示系统当前启用的配方ID；

F:倒计时桶槽制程时间；

G:显示桶槽配方设定的制程时间；

H:显示桶槽名称，此栏三种状态显示，灰色如SC槽代表当前BYPASS状态，黄色代表不具备制程条件，黑色代表可正常生产槽；另外，带下划线如HDIW2槽代表是两只ROBOT的交接槽，其他代表是普通槽；同时点击桶槽可进入相应桶槽手动操作页面；

I:显示LOADER侦测晶舟的sensor，共有三组六个，用来侦测是否存在晶舟，有晶舟的时候显示绿色，无晶舟的时候显示灰色，同样在DRY和UNLOAER也分布有这样的sensor；

J:显示每个桶槽已经进入的RUN数统计；

K:显示桶槽SHUTTER状态，共有三种状态：开启、关闭、未知，上图显示极为未知状态，代表正在开启或关闭过程中，如果长时间出于此模式，则需要查找机构电控问题；

L:显示ROBOT，机台共有三只ROBOT，此页面会显示ROBOT的运动，如取料、置料、移动等等；同时点击ROBOT可进入相应ROBOT手动操作、位置设定页面；

M:显示LOADER自动门的开启关闭状态，UNLOADER也同样如此；

O:显示PLC与PC通讯状态，红灯在不间断闪烁，表示通讯正常，长时间不闪烁代表通讯中断，需要检查物理通讯线路和PC\PLC程式是否在正常运行；

P:按钮操作区，分别是控制日光灯、控制蜂鸣器、控制进料、RUN数清零；

Q:显示系统操作按钮，分别是开启配方管理页面、用户管理页面、查询历史数据、设定系统参数、显示系统帮助文档，以及用户登入、登出、关闭系统等操作；

R:显示制程槽药水的可使用时间和可生产的RUN数倒计时；

S:显示制程槽的当前摄氏温度；

T:显示蚀刻槽自动换酸的状态；
