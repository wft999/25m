Inline桶槽手动操作

在主页上面，点击机台Inline桶槽部分，出现画面如下图：

(1)数据查看，本页可查看到桶槽液位、当前温度数值、温度历史趋势、循环泵、加热器等的开闭状态；

(2)加热操作，点击“加热”按钮，可执行加热开启或关闭动作；

(3)循环泵操作，点击“循环泵”按钮，可启动停止循环泵；
