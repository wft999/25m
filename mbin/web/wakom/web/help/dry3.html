DRY3手动操作

在主页上面，点击机台DRY3桶槽部分，出现画面如下图：

(1)数据查看，本页可查看到当前温度数值、温度历史趋势、循环泵等开闭状态；

(2)帐料操作，点击“帐料”按钮，可对桶槽帐料进行添加删除操作，注意：操作之后，请确保帐料状态与实际一致，否则会造成撞机；；

(3)加热操作，点击“加热”按钮，可执行加热开启或关闭动作；

(4)循环泵操作，点击“循环泵”按钮，可执行循环泵开启关闭动作；

(5)上盖操作，点击“上盖”按钮，可控制上盖的开启和闭合；
