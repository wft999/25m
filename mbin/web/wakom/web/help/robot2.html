Robot2管理

在主页面,点击中间那个ROBOT图标，出现画面如下图：

(1)查看桶槽位置，选择桶槽列表下拉框，可查看HDIW2、TEX3-1、TEX3-2、TEX4-1、TEX4-2、HDIW3、HDIW4-1、HDIW4-2、HF-1、HF-2、HDIW5等槽的位置，每个位置分为夹爪脱钩和夹爪挂人位置；

(2)变更桶槽位置，由下拉框选择需要位置变更的桶槽，然后点击脱钩位置或者挂人位置的变更按钮，系统将ROBOT当前位置覆盖到相应桶槽的脱钩位置或者挂人位置，执行此命令，需要慎重，确认无误才能执行，不然会造成ROBOT撞机；

(3)变更安全高度,安全高度是ROBOT在水平方向桶槽间移动的时候所设定的Y方向位置，点击变更按钮，系统将会把ROBOT当前Y位置覆盖到安全高度，执行此命令，需要慎重，确认无误才能执行，不然会造成ROBOT撞机；

(4)手动输入，选择Manual Input选项，可直接编辑脱钩位置、挂人位置、安全高度，主要用于在校正ROBOT位置的时候进行微调，执行此命令，需要慎重，确认无误才能执行，不然会造成ROBOT撞机；；

(5)移动ROBOT到桶槽上方，先在列表框选择目标桶槽，然后点击“移栽校正位”按钮，ROBOT会自动移动到目标桶槽所设定的挂人位置，注意：需要正确校正过目标桶槽的位置之后，才能执行此动作，以免撞机；

(6)移动ROBOT到桶槽内取料，先在列表框选择目标桶槽，然后点击“夹爪取料”按钮，ROBOT会自动移动到目标桶槽执行取料动作，注意：需要正确校正过目标桶槽的位置之后，才能执行此动作，以免撞机；

(7)移动ROBOT到桶槽内置料，先在列表框选择目标桶槽，然后点击“夹爪置料”按钮，ROBOT会自动移动到目标桶槽执行置料动作，注意：需要正确校正过目标桶槽的位置之后，才能执行此动作，以免撞机；

(8)ROBOT原点复归，点击“原点复归”按钮，ROBOT会自动做原点复归动作；

(9)ROBOT JOG操作，短按四个方向箭头，可进行寸动操作；其中双箭头代表高速、单箭头代表低速；

(10)ROBOT CHUCK操作，点击“Chuck”按钮，可对ROBOT Chuck进行打开关闭操作；

(11)ROBOT TEACH操作，点击“TEACH”按钮，可打开关闭TEACH BOX；

(12)ROBOT 帐料操作，点击“帐料”按钮，可对ROBOT 帐料进行添加删除操作，注意：操作之后，请确保帐料状态与实际一致，否则会造成撞机；
