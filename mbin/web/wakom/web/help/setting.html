系统设定

在主页面左下角，点击系统设定按钮，出现画面如下图：

(1)Chuck Lock启动，点选此选项后，PLC在取料置料作业过程中中会检测抓手上是否有晶舟；

(2)门InterLock启动，点选此选项后，系统会侦测机台门的开闭并进行安全保护；

(3)Exahaust侦测启动，点选此选项后，系统将对Exahaust进行报警侦测；

(4)SafeHeight模式启动，点选此选项后，ROBOT将仅在安全高度移动，用于机构测试；

(5)IPA BUFFER TANK模式启动；

(6)排放延时设定，制程槽在进行排放时，当桶槽LL液位消失时，系统会使用此设定来延时排放，以保证桶槽内完整排空；

(7)权限设定，系统分三种用户：操作工、制程工程师、设备工程师，权限分为六种权限：管理配方、管理账号、操作桶槽、操作夹爪、修改设定、退出系统；可分别为三种用户授予六种不同权限；

(8)点击存档启用按钮，系统将保存设定到硬碟并启用；
